Tipo de prueba Descripción Ejemplo
Pruebas unitarias Verificar el correcto funcionamiento de un módulo o unidad de código Probar una función que devuelve el promedio de una lista de números
Pruebas de integración Verificar que los diferentes módulos o unidades de código funcionan correctamente cuando se integran Comprobar que una página web se carga correctamente cuando se accede a ella desde diferentes navegadores
Pruebas de sistema Verificar que el sistema completo funciona correctamente y cumple con los requisitos especificados Comprobar que un sistema de reservas de vuelos permite realizar reservas, cancelaciones y modificaciones de vuelos correctamente
Pruebas de aceptación Verificar que el software cumple con los requisitos del cliente y se comporta como se espera Comprobar que un sistema de compras en línea permite realizar compras de forma fácil y segura, y que el proceso de pago funciona correctamente
